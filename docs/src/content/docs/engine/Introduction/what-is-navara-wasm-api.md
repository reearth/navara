## What is navara_wasm_api?

navara_wasm_api is a WebAssembly-based API that provides mathematical calculations and utility functions needed for developing 3D geospatial applications.

## Key Features

- GIS coordinate transformations and projection calculations
- Geometric operations in 3D space

## Characteristics

- **High Performance**: Fast computation powered by WebAssembly
- **Ease of Use**: Includes TypeScript type definitions
- **Accuracy**: High-precision calculations compliant with WGS84
