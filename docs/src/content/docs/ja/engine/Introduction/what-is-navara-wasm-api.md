## navara_wasm_api とは？

navara_wasm_api は、3D 地理空間アプリケーション開発に必要な数学的計算とユーティリティ機能を提供する WebAssembly ベースの API です。

## 主な機能

- GIS 系の座標変換や投影計算
- 3D 空間における幾何演算

## 特徴

- **高性能**: WebAssembly による高速計算
- **使いやすさ**: TypeScript 型定義付き
- **正確性**: WGS84 準拠の高精度計算
